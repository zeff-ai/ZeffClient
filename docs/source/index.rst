ZeffClient's Documentation
==========================

ZeffClient is a collection of libraries and tools to make working with
Zeff Cloud API online systems easier.

   - :doc:`zeff` simplifies the creation, modification, and uploading
     of records to Zeff Cloud API.

   - :doc:`zeff.record` is for building records that may be validated
     or uploaded to Zeff Cloud API.

   - :doc:`zeff.cli` simplifies experimentation with Zeff Cloud API.
